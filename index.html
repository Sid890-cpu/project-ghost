// How it works

Four steps.
Zero noise.

01
🌐
Fetch
Ghost fetches any URL — HTML pages, RSS feeds, JSON APIs. Blocked sites automatically fall back to their public RSS endpoints.
02
🧹
Clean
All scripts, styles, ads, and navigation are stripped. Raw text is parsed into structured headlines — cutting content by up to 50%.
03
🧠
Extract
Groq LLM extracts entities, impact scores, summaries, and business intent. Priority is calculated server-side — never hallucinated.
04
⚡
Signal
A clean Ghost Signal is returned and stored in Supabase. Your agent reads structured JSON — not a wall of text — and acts instantly.
